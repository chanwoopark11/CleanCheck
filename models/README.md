# 🧠 Models

이 디렉토리는 CleanCheck 프로젝트에서 사용된 학습 완료 모델을을 저장한 공간입니다. WHO 6단계 손씻기 동작 인식을 위한 YOLOv8 기반 모델이 포함되어 있으며, 각각 다른 데이터셋과 학습 조건으로 훈련되었습니다.

## 📁 모델 구성

### 📂 Train_v1
- **데이터 구성**:  
  - Train: 4,652장  
  - Validation: 582장  
  - Test: 582장  
  - **총 5,812장**
- **포함 내용**:
    - `last.pt`: 마지막 에폭까지 학습한 모델  
    - `best.pt`: 가장 우수한 성능을 보인 모델 (실제 사용 추천)
